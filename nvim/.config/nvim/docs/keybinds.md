# Neovim Keybinds

**Leader:** `Space`

---

## Geral
- `<leader>w` → salvar  
- `<leader>q` → sair  
- `<leader>h` → limpar busca

---

## Telescope
- `<leader>f` → buscar arquivos com preview
- `<leader>g` → live grep
- `<leader>b` → buffers
- `<leader>hh` → help tags (documentação do Vim/Neovim)

---

## Harpoon
- `<leader>a` → marcar arquivo  
- `<leader>ra` → remover atual  
- `<leader>rc` → limpar tudo  
- `<leader>e` → menu rápido  
- `<leader>1` a `<leader>4` → ir para 1–4  
- `<leader>p` → anterior  
- `<leader>n` → próximo

---

## LSP (Ruby)
- `gd` → definição  
- `K` → hover  
- `<leader>r` → renomear  
- `gl` → diagnóstico  
- `<leader>gr` → referências  
- `<leader>gD` → declaração

---

## Autocompletar (Insert)
- `<C-Space>` → abrir menu  
- `<CR>` → confirmar  
- `<Tab>` / `↓` → próximo  
- `<S-Tab>` / `↑` → anterior
